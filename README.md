# Spotify Playlist Manager

**Real-time playlist management web app**

Spotify Playlist Manager allows users to log in with their Spotify account, view their playlists, and interact with them in a more powerful way than Spotify’s native interface.

Users can compare playlists, remove tracks, and drag and drop songs between playlists.

## Tech Stack

- **Frontend:** React + Vite
- **API:** Spotify Web API
- **Backend:** Node.js, Express
- **Languages:** JavaScript
- **Hosting:** Render, Vercel

## Features

- Real-time playlist fetching using Spotify API
- Playlist comparison interface
- Song removal with confirmation modal
- OAuth authentication with secure cookie handling
- Backend API integration with Express
